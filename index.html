gimpmode by pft

GIMP <-> Emacs interaction

Emacs mode for developing scheme (script-fu) scripts for the GIMP. Interaction with the GIMP as subprocess or as a client. Smart and fuzzy code completion . Documentation echoing. Hypertext help system with history.

Dependencies

GNU Emacs and GIMP (>= 2.4)

Install

Start Emacs and run the installation script: M-x load-file /path/to/gimp-install.el RET And follow the directions given there. (It will direct you to put something like the following in your load file and evaluate it: (load "/path/to/gimp-mode/gimp-init.el") ...that file basically sets up everything so that stuff is only loaded when needed.)
